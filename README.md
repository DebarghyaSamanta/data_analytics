# data_analytics
TASK 1: To perform EDA and build a univariate regression model using only NumPy and Pandas without using sklearn.
Steps Performed :
1. Exploratory Data Analysis : Checked correlations and Selected the most correlated feature with target variable
2. Feature Selection : Selected one feature (highest correlation with output)
3. Model Implementation : Built linear regression model: y = mX + b and Implemented Gradient Descent using loop
4. Optimization : Computed gradients: dm (slope update) , db (intercept update) , Updated parameters iteratively
5. Calculated : SS Total , SS Residual , R² Score
6. Visualization : Plotted Loss vs Iterations curve
